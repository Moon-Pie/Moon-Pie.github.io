Prufer Code

Prufer 的构造

每次选择一个编号最小的叶结点并删掉它，然后在序列中记录下它连接到的那个结点。重复 n−2n-2n−2 次后就只剩下两个结点，算法结束。

prufer序列的两个性质

  • 在构造完 Prufer 序列后原树中会剩下两个结点，其中一个一定是编号最大的点 nnn。
  • 每个结点在序列中出现的次数是其度数减 111。（没有出现的就是叶结点）

用Prufer序列重构一棵树

先将每个点的度数算出来。对于Prufer序列的第一个数，就是当前编号最小的叶子的父亲。然后，删掉这个编号最小的叶子，并维护度数。重复 n−2n-2n−2 次后，序列中的数用完，必然仍有两个点度数为 111，连边即可。

其实构造prufer序列和用prufer序列重构树都不太重要了。就只要知道一棵大小为 nnn 的树可以和一个长度为 n−2n-2n−2 的序列一一对应就行了。
