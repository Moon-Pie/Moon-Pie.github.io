2021-08-22

内容

max⁡(S)=∑T⊆S(−1)∣T∣+1min⁡(T)\max(S) = \sum_{T \subseteq S}(-1)^{|T|+1}\min(T) max(S)=T⊆S∑​(−1)∣T∣+1min(T)

假设我们能很轻松地求出任意集合的 min⁡(S)\min(S)min(S) ，但是我们不会(或者很难)求出任意集合的 max⁡(S)\max(S)max(S) 。

Min-Max容斥就是通过一种奥妙重重的方式把,Min转换成Max(或者反之)。

附：Min-Max容斥定理在期望下也成立。

也就是说 E(max⁡(S))=∑T⊆S(−1)∣T∣+1E(min⁡(T))E(\max(S))=\sum\limits_{T\subseteq S}(-1)^{|T|+1}E(\min(T))E(max(S))=T⊆S∑​(−1)∣T∣+1E(min(T))
这是非常有用的，因为期望下的 max⁡\maxmax 和 min⁡\minmin 是很难求的。

证明

设 AkA_kAk​ 为 UUU 中的第 kkk 大的元素。设 min⁡(T)=Ai\min(T) = A_imin(T)=Ai​。

  1. i=1i = 1i=1
    如果 min⁡(T)=A1\min(T) = A_1min(T)=A1​，就只有一种情况 T={A1}T = \{A_1\}T={A1​}。贡献为 (−1)2A1=A1(-1)^2A_1 = A_1(−1)2A1​=A1​
  2. i>1i > 1i>1
    A1,A2,…,Ai−1A_1,A_2,\dots,A_{i-1}A1​,A2​,…,Ai−1​ 可以出现在 TTT 中所以有 2i−12^{i-1}2i−1 种 TTT。不难发现，有 2i−22^{i-2}2i−2 种选法使 ∣T∣|T|∣T∣ 是偶数，有 2i−22^{i-2}2i−2 种选法使 ∣T∣|T|∣T∣ 是奇数。所以贡献为 000。

拓展（求 k_max\mathrm{k\_max}k_max）

  • 如果拓展到一般情况，求第 kkk 大的怎么办？

我们还是考虑容斥，并根据上面的特殊情况可以得到：

k_max(S)=∑T⊆Sf(∣T∣)min⁡(T)\mathrm{k\_max}(S) = \sum_{T\subseteq S}f(|T|)\min(T) k_max(S)=T⊆S∑​f(∣T∣)min(T)

fff 是一个函数，需要我们构造出来。根据上面证明的思路：

如果 min⁡(T)=Ap\min(T) = A_pmin(T)=Ap​，那么 A1,A2,…,Ap−1A_1,A_2,\dots,A_{p-1}A1​,A2​,…,Ap−1​ 这 p−1p-1p−1 个元素可选可不选。那么 ApA_pAp​ 前的系数就是：

∑i=0p−1(p−1i)f(i+1)\sum_{i=0}^{p-1}\binom{p-1}{i}f(i+1) i=0∑p−1​(ip−1​)f(i+1)

所以构造的 fff 需要满足：

∑i=0p−1(p−1i)f(i+1)=[p=k]\sum_{i=0}^{p-1}\binom{p-1}{i}f(i+1) = [p = k] i=0∑p−1​(ip−1​)f(i+1)=[p=k]

喜闻乐见的 二项式反演：

∑i=0p−1(p−1i)f(i+1)=[(p−1)+1=k]\sum_{i=0}^{p-1}\binom{p-1}{i}f(i+1) = [(p-1)+1 = k] i=0∑p−1​(ip−1​)f(i+1)=[(p−1)+1=k]

f(i+1)f(i+1)f(i+1) 看成一个关于 iii 的函数，左边看成是一个关于 p−1p-1p−1 的函数。

然后就得到了一般形式的式子：

k_max(S)=∑T⊆S(−1)∣T∣−k(∣T∣−1k−1)min⁡(T)\mathrm{k\_max}(S) = \sum_{T\subseteq S}(-1)^{|T|-k}\binom{|T|-1}{k-1}\min(T) k_max(S)=T⊆S∑​(−1)∣T∣−k(k−1∣T∣−1​)min(T)

[HAOI2015]按位或

题意

一开始你有一个数字 000。每秒钟你会得到一个在 [0,2n−1][0,2^n-1][0,2n−1] 的数字，第 iii 个数出现的概率是 pip_ipi​。问期望多少秒后，你手上的数字变成 2n−12^n-12n−1。

题解

可以按位考虑，并将每个数看成一个数集。

设 aia_iai​​ 为第 iii​ 个二进制位变为 111​ 的时间。A={a0,a1…,an−1}A = \{a_0,a_1\dots,a_{n-1}\}A={a0​,a1​…,an−1​}​。

设 P(S)P(S)P(S) 表示选中的数是 SSS 集合的子集的概率。这个用高维前缀和或DWT做。

然后，设 UUU 为全集。就得到一个人畜无害的式子：

E(min⁡(S))=∑k=1+∞k⋅P(U⊕S)k−1P(S)E(\min(S)) = \sum_{k=1}^{+\infty}k\cdot P(U\oplus S)^{k-1}P(S) E(min(S))=k=1∑+∞​k⋅P(U⊕S)k−1P(S)

[Luogu P4707] 重返现世

题意

有 nnn 个不同的物品，每秒有 pim\cfrac{p_i}{m}mpi​​ 的概率出现 iii 物品。求收集了 kkk 个物品的期望时间。

n≤1000,0≤n−k≤10,∑pi=m,1≤m≤10000n\le 1000, 0\le n-k \le 10,\sum_{p_i} = m,1\le m\le 10000 n≤1000,0≤n−k≤10,pi​∑​=m,1≤m≤10000

题解

设 SSS 为一个物品的集合。将一个物品的权值设为其第一次出现的时间。

那么 E(min⁡(T))\mathbb{E}(\min(T))E(min(T)) 就是 TTT 集合中出现一个的期望时间。每次得到 TTT 集合中的元素的概率为 ∑i∈Tpim\frac{\sum_{i\in T} p_i}{m}m∑i∈T​pi​​，那么 E(min⁡(T))=m∑i∈Tpi\mathbb{E}(\min(T)) = \frac{m}{\sum_{i\in T} p_i}E(min(T))=∑i∈T​pi​m​。

不难发现 E(k_min(U))E(\mathrm{k\_min}(U))E(k_min(U)) 就是答案（UUU 是全集）。我们令下文中 k←n−k+1,k≤11k\leftarrow n-k+1,k\le 11k←n−k+1,k≤11。

E(k_max(U))=∑T⊆U(−1)∣T∣−k(∣T∣−1k−1)E(min⁡(T))\mathbb{E}(\mathrm{k\_max}(U)) = \sum_{T\subseteq U}(-1)^{|T|-k}\binom{|T|-1}{k-1}\mathbb{E}(\min(T)) E(k_max(U))=T⊆U∑​(−1)∣T∣−k(k−1∣T∣−1​)E(min(T))

显然不能直接 O(2n)O(2^n)O(2n)。由于数据的特殊性：k,mk,mk,m 都不大，考虑大力 DPDPDP。

有一种很naive 的 O(n2m)O(n^2m)O(n2m) 做法：

为了计算答案，设 fi,j,kf_{i,j,k}fi,j,k​ 表示考虑到前 iii 个物品，满足一下两个条件的集合 TTT 的个数：

  • ∑i∈Tpi=j\sum_{i\in T}p_i = j∑i∈T​pi​=j
  • ∣T∣=k|T| = k∣T∣=k

转移：

fi,j,k=fi−1,j−1,k−pi+fi−1,j,kf_{i,j,k} = f_{i-1,j-1,k-p_i} + f_{i-1,j,k} fi,j,k​=fi−1,j−1,k−pi​​+fi−1,j,k​

所以我们需要优化掉一个维度。事实上，我们记录 ∣T∣=k|T| = k∣T∣=k，是为了计算 (−1)∣T∣−k(∣T∣−1k−1)(-1)^{|T|-k}\binom{|T|-1}{k-1}(−1)∣T∣−k(k−1∣T∣−1​)，那我们为什么不直接将记录方案数 变成记录所有 TTT 集合的 (−1)∣T∣−k(∣T∣−1k−1)(-1)^{|T|-k}\binom{|T|-1}{k-1}(−1)∣T∣−k(k−1∣T∣−1​) 的和呢？

设 fi,jf_{i,j}fi,j​ 表示考虑到前 iii 个物品，∑i∈Tpi=j\sum_{i\in T}p_i = j∑i∈T​pi​=j 的所有的集合 TTT 的 (−1)∣T∣−k(∣T∣−1k−1)(-1)^{|T|-k}\binom{|T|-1}{k-1}(−1)∣T∣−k(k−1∣T∣−1​) 的和。

转移还是有两种，第 iii 个选或不选。不选就是直接继承。如果选了 iii，那么 TTT 集合的大小加一。

(−1)∣T∣−k(∣T∣−1k−1)⟶(−1)∣T∣+1−k(∣T∣k−1)=(−1)∣T∣−k+1((∣T∣−1k−1)+(∣T∣−1k−2))=−(−1)∣T∣−k(∣T∣−1k−1)+(−1)∣T∣−(k−1)(∣T∣−1k−2)\begin{aligned} (-1)^{|T|-k}\binom{|T|-1}{k-1} \longrightarrow &(-1)^{|T|+1-k}\binom{|T|}{k-1} \\ =& (-1)^{|T|-k+1}\left(\binom{|T|-1}{k-1}+\binom{|T|-1}{k-2}\right) \\ =& -(-1)^{|T|-k}\binom{|T|-1}{k-1} + (-1)^{|T|-(k-1)}\binom{|T|-1}{k-2} \end{aligned} (−1)∣T∣−k(k−1∣T∣−1​)⟶==​(−1)∣T∣+1−k(k−1∣T∣​)(−1)∣T∣−k+1((k−1∣T∣−1​)+(k−2∣T∣−1​))−(−1)∣T∣−k(k−1∣T∣−1​)+(−1)∣T∣−(k−1)(k−2∣T∣−1​)​

分出来的两项都和我们原来记录的很像。实际上，我们只需要再记录一个 kkk 就可以有 O(nmk)O(nmk)O(nmk) 的DP了。

转移就是：

fi,j,k=fi−1,j,k+fi−1,j−pi,k−1−fi−1,j−pi,kf_{i,j,k} = f_{i-1,j,k} + f_{i-1,j-p_i,k-1} - f_{i-1,j-p_i,k} fi,j,k​=fi−1,j,k​+fi−1,j−pi​,k−1​−fi−1,j−pi​,k​
